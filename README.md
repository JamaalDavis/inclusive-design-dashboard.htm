# Inclusive Design Intelligence Dashboard

Static prototype and design artifacts for the Inclusive Design Intelligence Dashboard.

The dashboard tells a June 2026 story about inclusive design shifting from inspiration and advocacy into product infrastructure: compliance pressure, AI recovery, design-system governance, and neuroinclusive UX.

## What This Includes

- A standalone HTML dashboard prototype.
- A GitHub Pages root link that opens the dashboard.
- A separate overview page with project context and artifact links.
- Accessibility-patterned chart bars so the data is not encoded by color alone.
- Research framing around inclusive design, AI accessibility, compliance, design systems, and neuroinclusion.
